## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+
- Android Studio (for mobile development)
- Solana CLI tools
- MongoDB instance
- Redis instance

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-org/nexhub.git
   cd nexhub
   ```

2. **Install backend dependencies**
   ```bash
   cd challanger
   npm install
   ```

3. **Install mobile app dependencies**
   ```bash
   cd mobile_app/android
   ./gradlew build
   ```

4. **Set up environment variables**
   ```bash
   # Copy and configure environment files
   cp challanger/.env.example challanger/.env
   # Configure your MongoDB, Redis, and AI API keys
   ```

5. **Start the backend services**
   ```bash
   cd challanger
   npm run dev
   ```

6. **Run the mobile app**
   ```bash
   cd mobile_app/android
   ./gradlew installDebug
   ```
